<Random Team Generator>

This bot is for communities or groups of friends that like to organize games to play and choose randomly the teams

the bot splits users in a voice channel into 2 or more teams randomly

Some commands like !mw2 show a map to play on.

Commands

!help - shows all Commands

!map - shows maps from different games

!rtg - makes an embed with the information on the teams

More Commands

!mw2, !lol, !csgo, !ow(overwatch), !vr(valorant).

Info

session: you can only have one session for each voice channel

More Future Features

I have more ideas to put like creating for each team a voice channel but i need more knowledge on the discord.js api

Disclaimer: this bot is not finished, commands like !end, !cache, !reload and !mu don't work
